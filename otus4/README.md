ДЗ: Работа с mdadm

Работа с mdadmin

    1. Создаем в папке Vagrant подпапку для урока
    2. Копируем в нее файлы Vagrantfile, raid.sh
    3. Vagrant up

Чтобы убедится, что имя девайса рейд после ребута остается старым - необходимо:
1. vagrant ssh
2. sudo reboot
3. lsblk
